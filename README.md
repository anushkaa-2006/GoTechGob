# 🚀 GoTechGob

GoTechGob is a web-based career guidance and placement preparation platform developed using **PHP** and **MySQL**. It is designed to help students and job seekers by providing curated learning resources, career roadmaps, placement preparation materials, professional documents, and the latest job opportunities—all in one place.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/anushkaa-2006/GoTechGob.git
```

### 2. Move the Project

Copy the project folder into:

```
C:\xampp\htdocs\
```

### 3. Start XAMPP

Start the following services:

- Apache
- MySQL

### 4. Open the Project

Open your browser and visit:

```
http://localhost/GoTechGob
```

## 🎯 Purpose

The goal of GoTechGob is to provide students with a single platform where they can:

- Stay updated with the latest technology trends.
- Follow structured career roadmaps.
- Prepare for placements and interviews.
- Access professional resume and cover letter resources.
- Explore internship and job opportunities from leading companies.
